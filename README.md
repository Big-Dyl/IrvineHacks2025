# IrvineHacks2025

Our hackathon project for IrvineHacks 2025

## Run

Development frontend:
```sh
cd ./frontend/
npm install
npm run dev
```

Development backend:
```sh
cd ./backend/
npm install
node index.js
```

## Technologies

 Frontend:
- Vite and React
- Tailwind CSS

Backend:
- Node.js
- Express
- Socket
